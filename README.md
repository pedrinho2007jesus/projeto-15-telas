## Sobre o projeto
O projeto 15 tela consiste em um sistema com páginas de administração, login, cadastro, perfil e gerenciamento de postagens e comentários.

## Desenvolvimento
O projeto foi desenvolvido em equipe, com todos os integrantes reunidos em call durante a criação das páginas, organização das ideias e desenvolvimento do sistema.
